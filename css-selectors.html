Combinator Selectors

The group of selector selects all the HTML elements with the same style

Look at the following code(h1, h2,.... and p elements have the samae style definitions)

                    
                        Syntax:
                        h1, h2, h3, h4, h5, h6, p{
                            color: #000000;
                            background-color: #ffffff
                            text-align: center
                        }

Type of Combinator Select

  1. Descendant Selector (space)
  2. Child Selector ( > )
  3. Adjacent Selector ( + )
  4. General Selector (~)
