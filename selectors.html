CSS Selectors

We can divide CSS selectors into 5 categories

  1. Simple Selectors
  2. Combinator Selectors
    • Descendant Selector (Space)
    • Child Select (>)
    • Adjacent Sibiling Selector (+)
    • General Sibiling Selector (~)
  3. Pseado-Class Selectors
  4. Pseado-Element Selectors
  5. Attribute Selectors
